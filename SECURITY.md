# Security Policy

## Supported Versions

As this is an educational exercise for the Coding Academy, security updates are only provided on a best-effort basis for the active branch.

| Version | Supported          |
| ------- | ------------------ |
| main    | Best-effort (Active)|
| < 1.0   | No                 |

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please do **not** open a public issue. Instead, report it privately to resolve the issue responsibly.

To report a vulnerability, please reach out directly to the repository owner, Aviad Benhamo.

Thank you for helping keep this project secure!
